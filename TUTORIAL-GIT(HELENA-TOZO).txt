Download do git bash no windows:

Esse é um terminal que funciona tantos com os comandos do cmd padrao
do windows quanto comandos do linux.

Vá ao seguinte site baixe a versao mais "adequada" ao seu windows:

https://git-scm.com/download/win

Na instalação vai dando accept(acho que nao tem muita coisa que possa acrescentar 
mas vc pode ir lendos as opções se quiser).



//comandos

Ao instalar vc precisará configurar o usuário global.

Abra o git bash e escreva os comandos:

$ git config --global user.name "seu nome de usuario"
$ git config --global user.email nome@email.com //email que vc usou para fazer sua conta no github.


Aperte o botao """ clone or download""""  e copie a URL dada.

No seu computador escolha uma pasta para ficar os códigos  do repositório.

No git bash caminhe até essa pasta (EXEMPLO: para ir para downloads faça $ cd "D:\Downloads").
            
***DICA: pelo gerenciador de arquivos é possível copiar o caminho da pasta desejada.

ao escolher a pasta digite o comando:

$ git clone URL

Assim os arquivos do repositório serão copiados para o seu computador.

Para adicionar algum arquivo apenas arraste o arquivo desejado para a pasta com o nome o repositório(tb-pds2-g19)
digite o comando :

$ git add .
//é com o ponto mesmo

Serão adicionados todas as mudanças feitas na pasta.

Em seguida digite o comando :

$ git commit -m "nome do commit"

"nome do commit" é algum nome que identifique a mudança feita no repositorio .(Por exemplo foi adicionada uma função que embaralha as cartas).

Por fim para de fato mudar o repositorio faça o comando :

$ git push

Pronto a mudança do repositorio foi feita.
Caso a pasta ja esteja clonada mas os outros integrantes modifiquem o reposirio para atualizar a pasta na sua maquina basta fazer :

$ git pull

e apartir dai fazer as modificações que achar necessárias.

outros comnados::

$ git status 

//Mostra os arquivos que forma adicionados ou retirados no git add ou git commit.
